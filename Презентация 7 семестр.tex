\begin{frame}\frametitle{Метод бросания лучей}
Метод бросания лучей (или рейкастинг) - один из методов рендеринга в компьютерной графике,
при котором сцена строится на основе замеров пересечения лучей с визуализируемой поверхностью.

\begin{frame}\frametitle{Описание метода}
Основная идея:

\begin{figure}[h!]
\begin{center}
\includegraphics[width=10cm]{sl_7}\\ %основная идея
\end{center}
\end{figure}
\begin{minipage}{0.49 \linewidth}
$\overrightarrow{a_{i+1}} = a_i + \delta a$\\
$\overrightarrow{a_0} = A$, $\overrightarrow{a_n}=B$\\
$\overrightarrow{r_i} = \overrightarrow{a_i} - I$\\
\end{minipage}
\hfill
\begin{minipage}{0.49 \linewidth}
Алгоритм рейкастинга:\\
\textit{
для каждой $\overrightarrow{r_i}$:\\
~~~$l$ $\leftarrow$ расстояние до стены(I, $\overrightarrow{r_i}$)\\
~~~$h$ $\leftarrow$ расчитать высоту отрезка $l$\\
~~~отобразить отрезок $(i, h)$\\
}
\end{minipage}

\end{frame}
\begin{frame}\frametitle{Описание метода}
Используем алгоритм Цифрового Дифференциального анализатора(DDA) для нахождения расстояния до стены:\\
\begin{minipage}{0.49 \linewidth}
\begin{figure}[h!]
\begin{center}
\includegraphics[width=5cm]{sl_8}\\ %дда
\end{center}
\end{figure}
\end{minipage}
\hfill
\begin{minipage}{0.49 \linewidth}
\textit{цикл:\\
~~~если $l_x$ < $l_y$\\
~~~~~~$l_x$ $\leftarrow$ $l_x$ + $\delta x$\\
~~~иначе\\
~~~~~~$l_y$ $\leftarrow$ $l_y$ + $\delta y$\\
пока $М /[l_x/]/[l_y/]$ не станет $>0$\\
вернуть $L = \sqrt{l_x^2+l_y^2}$
}
\end{minipage}
\end{frame}
